# Bellcorp Personal Expense Tracker

A secure web app to manage daily finances: sign up, log in, add/edit/delete transactions, view dashboard summary and category breakdown, and explore transactions with search, filters, and load more.

## Tech Stack

- **Frontend**: React 18, React Router 6, Vite
- **Backend**: Node.js, Express
- **Database**: JSON file store (no native build; works on all platforms). You can swap in MongoDB or SQLite for production.
